ErsatzTV supports both music videos (with video tracks) and audio-only songs, allowing you to create dedicated music channels with custom visualizations and metadata display.

## Music Videos

Music videos are video files with artist and track metadata. They can be imported from:

- **Local libraries**: Music video folders with proper metadata
- **Plex**: Music video libraries
- **Jellyfin/Emby**: Music video collections

### Metadata Requirements

For best results, music videos should include:

| Field | Description | Source |
|-------|-------------|--------|
| Artist | Primary artist name | NFO, media server |
| Title | Song/track title | NFO, media server |
| Album | Album name (optional) | NFO, media server |
| Year | Release year (optional) | NFO, media server |
| Genres | Music genres | NFO, media server |
| Thumbnail | Video thumbnail/poster | Artwork files |

### Channel Configuration

When creating a music video channel, configure these settings:

#### Music Video Credits Mode

Controls how artist and song information is displayed during playback.

<Tabs>
  <Tab title="None">
    No on-screen credits or subtitles are generated. Use this if your music videos already have burned-in text or if you don't want any text overlay.
    
    ```
    ChannelMusicVideoCreditsMode: None
    ```
  </Tab>
  <Tab title="Generate Subtitles">
    ErsatzTV automatically generates subtitle tracks with artist and song information. These appear as standard subtitles that viewers can enable/disable.
    
    ```
    ChannelMusicVideoCreditsMode: GenerateSubtitles
    ```
    
    **Generated subtitle content includes:**
    - Artist name
    - Song title
    - Album name (if available)
    
    Configure subtitle settings:
    - **Credits Template**: Customize the subtitle format
    - **Subtitle Location**: Position of subtitles (bottom, top, etc.)
  </Tab>
</Tabs>

<Note>
Subtitles generated by ErsatzTV are embedded in the stream and work with all IPTV clients that support subtitle display.
</Note>

### Using Music Videos in Schedules

#### Classic Schedules

Add music videos to schedule items:

1. Create or edit a **Classic Schedule**
2. Add a **Schedule Item**
3. Set **Collection Type** to:
   - **Collection**: Select a music video collection
   - **Smart Collection**: Use a search query like `kind:musicvideo`
   - **Search Query**: Filter music videos directly

#### Playback Orders

Useful playback orders for music videos:

| Order | Description | Best For |
|-------|-------------|----------|
| **Shuffle** | Random playback | Variety channels |
| **Chronological** | By release date | Decade-themed channels |
| **Marathon (Artist)** | Group by artist | Artist spotlight blocks |
| **Marathon (Album)** | Group by album | Album playback |

<Tip>
Use **Marathon (Artist)** with shuffle to play several videos from one artist before moving to the next artist.
</Tip>

#### Smart Collections for Music Videos

Create smart collections to organize music videos:

```
# All music videos from a specific decade
kind:musicvideo AND year:[1990 TO 1999]

# Music videos by genre
kind:musicvideo AND genre:rock

# Specific artist
kind:musicvideo AND artist:"Taylor Swift"

# Combine collections
smart_collection:"80s Rock" OR smart_collection:"90s Alternative"
```

## Songs (Audio-Only)

ErsatzTV can play audio-only song files with generated video backgrounds and progress displays.

### Supported Audio Formats

- MP3 (`.mp3`)
- FLAC (`.flac`)
- M4A/AAC (`.m4a`)
- OGG Vorbis (`.ogg`)
- WAV (`.wav`)
- AIFF (`.aif`, `.aifc`, `.aiff`)

### Song Video Mode

Controls how audio-only songs are displayed when scheduled on a channel.

<Tabs>
  <Tab title="Default">
    Displays a static background image with album art and song metadata.
    
    ```
    ChannelSongVideoMode: Default
    ```
    
    **Display includes:**
    - Album artwork (if available)
    - Artist name
    - Song title
    - Album name
    - Static background
  </Tab>
  <Tab title="With Progress">
    Adds a progress bar showing playback position through the song.
    
    ```
    ChannelSongVideoMode: WithProgress
    ```
    
    **Display includes:**
    - Everything from Default mode
    - Animated progress bar
    - Time elapsed/remaining
  </Tab>
</Tabs>

### Customizing Song Backgrounds

ErsatzTV uses customizable background templates for song playback:

**Default templates location:**
```
/config/templates/
  _ArtistTitleAlbum_CenterTop.sbntxt
  _ArtistTitle_LeftMiddle.sbntxt
```

**Custom backgrounds:**

1. Navigate to your ErsatzTV config folder
2. Go to `config/artwork/songs/`
3. Replace `background.png` with your custom image
4. Restart ErsatzTV

<Warning>
Default files start with an underscore (e.g., `_background.png`). Create a custom version without the underscore (`background.png`) to override the default.
</Warning>

### Missing Album Art

When album art is not available:

1. Default placeholder is used from `config/artwork/songs/_default_album.png`
2. Create custom placeholder by adding `default_album.png` (no underscore)

## Creating a Music Channel

Complete example of a 24/7 music video channel:

### Step 1: Create Collections

```
# Create smart collections for different eras/genres
80s Music Videos: kind:musicvideo AND year:[1980 TO 1989]
90s Music Videos: kind:musicvideo AND year:[1990 TO 1999]
Rock Videos: kind:musicvideo AND genre:rock
```

### Step 2: Create Channel

1. Go to **Channels** > **Create Channel**
2. Set:
   - **Number**: 100
   - **Name**: "Music Video Classics"
   - **Music Video Credits Mode**: Generate Subtitles
   - **Song Video Mode**: With Progress (if including songs)

### Step 3: Create Schedule

1. Create a **Classic Schedule**
2. Add schedule items with time slots:

| Time | Collection | Playback Order |
|------|-----------|----------------|
| 12:00 AM | 80s Music Videos | Marathon (Artist), Shuffle |
| 6:00 AM | 90s Music Videos | Shuffle |
| 12:00 PM | Rock Videos | Marathon (Artist) |
| 6:00 PM | All Music Videos | Shuffle |

### Step 4: Configure FFmpeg Profile

For music channels, adjust these settings:

1. Go to **Settings** > **FFmpeg Profiles**
2. Edit or create a profile:
   - **Audio Codec**: AAC
   - **Audio Bitrate**: 192kbps or higher
   - **Video Bitrate**: Lower bitrate OK for static backgrounds (1-2 Mbps)
   - **Normalize Audio**: Enable for consistent volume

## EPG Customization for Music

Customize how music appears in the EPG guide:

### Music Video Template

Location: `/config/templates/musicVideo.sbntxt`

```xml
<programme start="{{ programme_start }}" stop="{{ programme_stop }}" channel="{{ channel_id }}">
  <title lang="en">{{ artist_title }}</title>
  <sub-title lang="en">{{ music_video_title }}</sub-title>
  {{ if music_video_has_album }}
    <desc lang="en">Album: {{ music_video_album }}</desc>
  {{ end }}
  <category lang="en">Music</category>
  {{ for genre in music_video_genres }}
    <category lang="en">{{ genre }}</category>
  {{ end }}
  {{ if music_video_has_artwork }}
    <icon src="{{ music_video_artwork_url }}" />
  {{ end }}
</programme>
```

<Tip>
Available template variables include `artist_title`, `music_video_title`, `music_video_album`, `music_video_year`, `music_video_genres`, and more. See the [EPG Customization guide](/guides/epg-customization) for details.
</Tip>

## Marathon Mode for Music

Marathon mode is perfect for music channels:

**Marathon Group By: Artist**
- Groups videos by artist
- Plays multiple videos from one artist before switching
- Optional shuffle of artists
- Optional shuffle of videos within each artist

**Marathon Group By: Album**
- Groups videos by album
- Plays full albums in sequence
- Great for album-focused channels

**Batch Size**:
- Set how many videos to play from each group before advancing
- Empty/0 = play all videos from the artist/album
- 3 = play 3 videos, then move to next artist/album

## Troubleshooting

### Music Videos Not Appearing

<AccordionGroup>
  <Accordion title="Check library scan">
    1. Go to **Media Sources** > **Libraries**
    2. Verify music video library was scanned successfully
    3. Trigger a deep scan if needed
  </Accordion>
  
  <Accordion title="Verify metadata">
    Music videos need proper artist and title metadata to be recognized. Check NFO files or media server metadata.
  </Accordion>
  
  <Accordion title="Search for music videos">
    Use the search bar with `kind:musicvideo` to see all imported music videos.
  </Accordion>
</AccordionGroup>

### Subtitles Not Showing

1. Verify **Music Video Credits Mode** is set to **Generate Subtitles**
2. Check that subtitle template exists: `/config/templates/default.ass.sbntxt`
3. Ensure IPTV client has subtitles enabled

### Songs Playing Without Video

1. Verify **Song Video Mode** is configured
2. Check logs for background generation errors
3. Ensure `/config/templates/` contains song templates

### Poor Audio Quality

Increase audio bitrate in FFmpeg profile:

1. **Settings** > **FFmpeg Profiles**
2. Set **Audio Bitrate** to 256kbps or 320kbps
3. Enable **Normalize Audio** for consistent volume
