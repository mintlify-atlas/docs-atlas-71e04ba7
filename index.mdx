<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your first channel streaming in minutes with these simple steps.
  </p>

  <Steps>
    <Step title="Install ErsatzTV">
      Download and install ErsatzTV using Docker, standalone binaries, or your preferred package manager.

      ```bash Docker
      docker run -d \
        --name ersatztv \
        -p 8409:8409 \
        -v /path/to/config:/root/.local/share/ersatztv \
        jasongdove/ersatztv:latest
      ```

      Access the web UI at `http://localhost:8409`
    </Step>

    <Step title="Connect Your Media">
      Add your media libraries through the web interface. ErsatzTV supports local folders, Plex, Jellyfin, and Emby.

      Navigate to **Settings** > **Media Sources** and configure your first library.
    </Step>

    <Step title="Create a Channel">
      Build your first channel by selecting content and creating a schedule.

      Go to **Channels** > **Add Channel**, set up your programming schedule, and start streaming immediately.
    </Step>

    <Step title="Watch Your Channel">
      Stream your channel using any IPTV player. Copy the M3U playlist URL from the channel details page.

      ```bash Get Playlist
      http://localhost:8409/iptv/channels.m3u
      ```

      Import this URL into VLC, Plex, Jellyfin, or any IPTV-compatible player.
    </Step>
  </Steps>
</div>
